# agentile (plugin)

Eleven skills that make an agent fluent in the Agentile methodology: spec-first, red-test-first, honesty-enforced development with a full documentation trail.

Start with the `agentile` master skill — it holds the 13 rules, the artifact map, the sprint lifecycle, and the routing table to the other ten:

`feature-spec` · `planset` · `test-plan` · `red-green` · `refactor-mutate` · `document-pass` · `retro` · `journal` · `essay` · `case-study`

Skills ship the canonical templates (SPRINT, DAILY, ADR, RETRO, JOURNAL, ESSAY, CASE_STUDY) alongside their SKILL.md. Templates are vendored from the `agentile` skeleton repo — treat them as read-only copies.

The method is portable: nothing in this plugin requires the Citrate federation. Federation-specific navigation lives in the sibling `citrate-federation` plugin.
