# citrate-federation (plugin)

Two skills for working inside the CitrateNetwork federated repositories:

- **`navigate`** — the orientation sequence, the truth hierarchy (what wins when sources disagree), tier semantics, the repo landscape by concern, and staleness rules.
- **`projects`** — running work across the federation: manifest + drift discipline (Rules 11–12), pin-bump propagation, handoff and work-order format, parallel agent dispatch, and the new-repo checklist.

Assumes the Agentile method (install the sibling `agentile` plugin); this plugin only adds what the federation layers on top.

Internal to CitrateNetwork — the file paths and repo names in these skills refer to the federation workspace.
